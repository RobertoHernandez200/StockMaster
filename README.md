# StockMaster - Gestor Inteligente de Inventarios

StockMaster es una aplicación móvil desarrollada en Kotlin enfocada en la administración inteligente de inventarios para tiendas y clientes.

La aplicación permite gestionar productos, empleados, permisos, proveedores y finanzas mediante un sistema organizado y fácil de usar. Además, los clientes pueden acceder al inventario de una tienda mediante un código único, consultar productos disponibles y crear listas de deseos para mejorar su experiencia de compra.

El proyecto fue desarrollado utilizando Firebase como base de datos y sistema de autenticación en tiempo real.

# Funcionalidades Principales

## Módulo de Tienda

La tienda cuenta con un sistema administrativo completo, siendo el núcleo principal de la aplicación.

### Autenticación
- Registro de tiendas.
- Inicio de sesión seguro.
- Generación automática de código único para cada tienda.

Este código permite conectar clientes con el inventario correspondiente de la tienda.

---

## Gestión de Inventario

La tienda puede administrar completamente sus productos mediante un sistema dinámico e inteligente.

### Funciones:
- Registrar productos.
- Editar productos.
- Eliminar productos.
- Control de stock en tiempo real.
- Organización del inventario.

---

## Gestión de Empleados

La aplicación permite crear cuentas para empleados y asignar permisos personalizados según el rol dentro de la tienda.

### Ejemplos de permisos:
- Ver productos.
- Registrar productos.
- Modificar inventario.
- Gestionar ventas.
- Acceder al módulo financiero.
- Administrar empleados.

Esto permite mejorar la seguridad y el control administrativo dentro del sistema.

---

## Gestión de Proveedores

La tienda puede registrar y administrar información de proveedores asociados al negocio.

### Funciones:
- Registrar proveedores.
- Guardar información de contacto.
- Consultar datos de proveedores.
- Organizar proveedores de la tienda.

Esta función permite mantener un mejor control sobre los distribuidores y contactos comerciales del negocio.

---

## Módulo Financiero

La aplicación cuenta con una sección financiera enfocada en el análisis y control del negocio.

### Incluye:
- Estadísticas de ventas.
- Ventas anuales.
- Reportes financieros.
- Información analítica.
- Visualización mediante gráficas dinámicas.

Este módulo ayuda a mejorar la toma de decisiones dentro de la tienda.

---

# Módulo de Cliente

Los clientes cuentan con su propio sistema de acceso independiente.

## Login y Registro
- Registro de clientes.
- Inicio de sesión personalizado.

---

## Acceso mediante Código de Tienda

El cliente debe ingresar el código único de la tienda para acceder a su inventario y productos disponibles.

### El cliente puede:
- Ver productos en stock.
- Consultar disponibilidad en tiempo real.
- Navegar por el catálogo de productos.

---

## Lista de Deseos

El cliente puede crear una lista personalizada con productos disponibles en stock.

### Funciones:
- Agregar productos.
- Eliminar productos.
- Actualizar la lista.
- Guardar productos favoritos.

Esta función permite organizar las compras antes de visitar la tienda.

# Instalación del Proyecto

##  Clonar el repositorio

```bash
git clone https://github.com/RobertoHernandez200/StockMaster.git
```

---

##  Abrir el proyecto en Android Studio

Abrir la carpeta del proyecto desde Android Studio.

---

##  Sincronizar Gradle

Esperar a que Android Studio sincronice automáticamente las dependencias del proyecto.

---

##  Ejecutar la aplicación

La aplicación puede ejecutarse de dos formas:

### Opción 1: Emulador Android
Iniciar un emulador desde Android Studio y ejecutar la aplicación.

### Opción 2: Generar APK
Generar el archivo APK desde Android Studio e instalarlo en un dispositivo Android físico.

---

## Disfrutar de la aplicación

Una vez instalada, la aplicación estará lista para utilizar todas sus funcionalidades.
